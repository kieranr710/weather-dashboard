# 🌦 Weather Dashboard

A responsive weather dashboard built using HTML, CSS and JavaScript.

## Features

- Search any city
- Current weather
- 5-Day Forecast
- Weather icons
- Current date & time
- Press Enter to search
- Responsive interface

## Technologies

- HTML5
- CSS3
- JavaScript
- Open-Meteo API
